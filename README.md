PalmLogistik - Sistem Manajemen RAM TBS
Aplikasi pencatatan transaksi Tandan Buah Segar (TBS) Kelapa Sawit berbasis web yang ringan, cepat, dan tersinkronisasi secara real-time.
Fitur Utama
 * Dashboard KPI: Pantau Tonase Masuk, Tonase Berangkat, dan Stok di RAM secara instan.
 * Kalkulasi Otomatis: Menghitung Tonase Bersih berdasarkan potongan (%) dan total bayar secara real-time.
 * Manajemen Biaya: Input manual biaya operasional (Solar, Gaji, dll).
 * Laporan Laba Rugi: Neraca keuangan otomatis untuk melihat performa bisnis.
 * Real-time Database: Data tersimpan aman di cloud menggunakan Firebase.
Cara Instalasi di GitHub Pages
 * Upload file index.html ke repository ini.
 * Masuk ke Settings > Pages.
 * Pilih branch main dan klik Save.
 * Akses link yang diberikan oleh GitHub.
Teknologi yang Digunakan
 * HTML5 & Tailwind CSS: Untuk antarmuka yang responsif.
 * JavaScript (Vanilla): Logika aplikasi.
 * Firebase Firestore: Database real-time.
 * Lucide Icons: Ikon antarmuka yang modern.
Dikembangkan untuk efisiensi operasional RAM Sawit.
